Guía rápida de Kustomer
  1. Ve a Kustomer → Settings → Webhooks → Add Webhook
  2. URL: https://tu-app.railway.app/kustomer/webhook
  3. Events: kustomer.conversation.message.send
  4. Copia el Webhook Secret generado → ponlo en KUSTOMER_WEBHOOK_SECRET
  5. Ve a Settings → API Keys → Add API Key → ponlo en KUSTOMER_API_KEY
